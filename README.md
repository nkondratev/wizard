## Description

With wizard you can install certain versions of golang, delete and switch
Use \<install> \<version> to download golang
Example:
```bash
wizard install 1.25.5
```

Use \<use> \<version> to switch version
Example:
```bash
wizard use 1.25.5
```

Use \<help> to all commands
Example:
```bash
wizard help
```

Use \<help> to all commands
Example:
```bash
wizard help
```

Use \<list> to see all local installed version of golang
Example:
```bash
wizard list
```

## Installation

Run install.sh
